# Student Marks Prediction System

Overview:
This is a beginner-level Machine Learning project that predicts student marks based on study hours, previous scores, and attendance.

---

About the project:

The project uses **Linear Regression**, a supervised learning algorithm, to predict marks.It demonstrates core concepts of **Fundamentals of
AI & ML** such as:
- Training a model
- Making predictions
- Understanding relationships between variables

---

Technologies Used:

- Python
- NumPy
- Scikit-learn

Features:

- Takes user input:
  -- Study hours
  -- Previous score
  -- Attendance
- Predicts student marks
- Simple and easy to understand

---

How to Run:

- Install python in your system
- Provide values when asked:
  --Study Hours
  --Previous Score
  --Attendance
- Try to check for errors if any.
